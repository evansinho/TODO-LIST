# TODO-LIST
A TO-DO list app built with javaScript.

## Built With

- HTML,
- CSS,
- JavaScript
- Bootstrap.
- Webpack.

### Prerequisites

- Node
- Npm

### Setup

- git clone `https://github.com/evansinho/TODO-LIST`
- cd TODO-LIST
- run npm install
- run npm run build
- Open the dist/index.html file in any web browser of your choosing.
